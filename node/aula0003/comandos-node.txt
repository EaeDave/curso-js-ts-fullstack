npm init -y (inicializa um projeto node gerando o package.json)
npm install [nome-do-pacote] (instala o pacote em sua última versão em node_modules e o coloca em dependencies)
npm update (atualiza todos os pacotes/módulos se houver algum desatualizado)
npm install -E (essa flag especifica a versão no qual deseja usar de um módulo/pacote, assim não atualizará em npm update)
npm install [nome-do-pacote] --save-dev (coloca o módulo/pacote em dev-dependencies)
npm install [nome-do-pacote] --save-prod (coloca o módulo/pacote em dependencies)
npm install [nome-do-pacote]@x.x.x (instala uma versão específica do módulo)
npm install [nome-do-pacote]@latest (instala a versão mais recente explicitamente do módulo)
npm uninstall [nome-do-pacote] (desinstala o módulo)
npm ls (lista as dependencies)
npm outdated (lista os módulos desatualizados)
